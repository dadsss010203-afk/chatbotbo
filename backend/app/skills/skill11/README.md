# Skill EMS Hoja 11

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 11 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill11/tools/calcular_hoja11_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: NACIONAL
- D: PROVINCIAL

# Skill EMS Hoja 13

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 13 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill13/tools/calcular_hoja13_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: EMS Nacional
- D: Ciudades Intermedias
- E: Trinidad - Cobija

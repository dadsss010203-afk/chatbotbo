# Skill EMS Hoja 12

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 12 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill12/tools/calcular_hoja12_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: América del Sur / Destino A
- D: América Central y El Caribe / Destino B
- E: América del Norte / Destino C
- F: Europa y Medio Oriente / Destino D
- G: África, Asia y Oceanía / Destino E

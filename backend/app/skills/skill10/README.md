# Skill EMS Hoja 10

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 10 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill10/tools/calcular_hoja10_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: América del Sur / Destino A
- D: América Central y El Caribe / Destino B
- E: América del Norte / Destino C
- F: Europa y Medio Oriente / Destino D
- G: África, Asia y Oceanía / Destino E

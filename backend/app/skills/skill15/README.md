# Skill EMS Hoja 15

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 15 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill15/tools/calcular_hoja15_json.sh --peso "800g" --columna "B"
```

Mapa de columnas:
- B: Sud América / Tarifa 1
- C: Centro América Y Florida EE.UU. / Tarifa 2
- D: Resto De EE.UU. / Tarifa 3
- E: Caribe / Tarifa 4
- F: Europa / Tarifa 5
- G: Medio Oriente / Tarifa 6
- H: África Y Asia / Tarifa 7

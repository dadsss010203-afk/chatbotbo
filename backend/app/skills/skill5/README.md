# Skill EMS Hoja 5

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 5 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill5/tools/calcular_hoja5_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: Local
- D: Nacional
- E: Depto.
- F: Prov.
- G: Destino Especial / Trinidad - Cobija
- H: Destino Especial / Riberalta - Guayaramerín

# Skill EMS Hoja 9

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 9 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill9/tools/calcular_hoja9_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: Local
- D: Nacional
- E: Prov. Dentro Depto.
- F: Prov. Depto. Prov.

# Skill EMS Hoja 16

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 16 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill16/tools/calcular_hoja16_json.sh --peso "800g" --columna "B"
```

Mapa de columnas:
- B: SUD AMERICA / Tarifa 1
- C: Centro América y Florida EE.UU. / Tarifa 2
- D: Resto de EEUU / Tarifa 3
- E: CARIBE / Tarifa 4
- F: EUROPA / Tarifa 5
- G: MEDIO ORIENTE / Tarifa 6
- H: AFRICA Y ASIA / Tarifa 7

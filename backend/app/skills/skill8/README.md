# Skill EMS Hoja 8

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 8 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill8/tools/calcular_hoja8_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: América Del Sur / Destino A
- D: América Central Y El Caribe / Destino B
- E: América Del Norte / Destino C
- F: Europa Y Medio Oriente / Destino D
- G: África, Asia Y Oceanía / Destino E

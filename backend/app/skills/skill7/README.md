# Skill EMS Hoja 7

Este paquete deja a tu agente listo para calcular tarifas de la Hoja 7 con un Excel reducido incluido.

## Integracion rapida

```bash
bash backend/app/skills/skill7/tools/calcular_hoja7_json.sh --peso "800g" --columna "C"
```

Mapa de columnas:
- C: Local
- D: Nacional
- E: Prov. Dentro Depto.
- F: Prov. Depto. Prov.
- G: Ciudad Especial / Trinidad - Cobija
- H: Ciudad Especial / Riberalta - Guayaramerín
